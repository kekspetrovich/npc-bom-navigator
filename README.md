# NPC BOM Navigator

Интерактивный инструмент для визуализации и анализа сложных производственных спецификаций (Bill of Materials).

## 🚀 Основные возможности
- **Интерактивный граф:** Визуализация структуры изделия любой вложенности с помощью React Flow.
- **Фокусировка на узлах:** При выборе элемента схема перестраивается, показывая только прямые связи (входящие и исходящие).
- **Фильтрация по этапам:** Возможность мгновенно перестроить всю схему относительно конкретного производственного этапа.
- **Умный расчет количеств:** Автоматический пересчет локального расхода материалов относительно выбранного узла.
- **Группировка данных:** Объединение покупных изделий по категориям или поставщикам для упрощения схемы.
- **Экспорт данных:** Быстрое копирование структурированной информации об элементе в буфер обмена.

## 🛠 Технологический стек
- React + TypeScript
- Vite
- React Flow (визуализация графов)
- Dagre (автоматическая раскладка узлов)
- Tailwind CSS (стилизация)
- Lucide React (иконки)

## 📦 Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/kekspetrovich/npc-bom-navigator.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```

## 🚢 Деплой на GitHub Pages
Проект настроен для автоматического деплоя. Просто выполните:
```bash
npm run deploy
```
Сайт будет доступен по адресу: `https://kekspetrovich.github.io/npc-bom-navigator/`
